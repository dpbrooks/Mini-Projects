# Mini-Projects
A repo for all my smaller projects(including old school work) that don't need a dedicated repo but I want to have stored on GitHub


## Java Projects
### Single File Projects
Compile with `javac project_name.java`
Run with `java project_name`

### Multi-File Projects
cd into project folder
Compile with `javac -d bin src/*.java`
Run with `java -cp bin project_name`
